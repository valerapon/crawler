# Super Crawler3000   
## Dependence:  
#### cURL  
## How to install libraies:  
ubuntu: <code>sudo apt-get install curl</code>  
fedora: <code>sudo dnf install curl</code>   
## How to compile:
<code>make</code>  
## How to run:  
<code>./var/main depthLevel reference</code>  
example: <code>./var/main 5 www.msu.kz</code>
